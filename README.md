# VentaFacil Online

## Descripción

VentaFacil Online es una Landing Page responsive diseñada para promocionar una plataforma de ventas en línea dirigida a emprendedores, pequeñas tiendas y clientes que buscan comprar productos de manera rápida y segura.

## Tecnologías utilizadas

- HTML5
- CSS3

## Diseño

Se utilizó una interfaz sencilla con una combinación de colores azul y blanco para transmitir confianza y modernidad. La estructura está organizada para facilitar la navegación del usuario.

## Características

- Landing Page responsive.
- Menú de navegación.
- Sección principal (Hero).
- Sección de beneficios.
- Formulario de contacto.
- Footer con información básica.

## Cómo ejecutar el proyecto

1. Descargar el proyecto.
2. Abrir la carpeta.
3. Ejecutar el archivo `index.html` en cualquier navegador web.
